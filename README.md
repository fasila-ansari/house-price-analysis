# house-price-analysis
Exploratory data analysis of a real housing dataset to understand factors influencing house prices using Python and data visualization.
## Dataset
The dataset used in this project is the "House Prices: Advanced Regression Techniques" dataset from Kaggle.
## Technologies Used
- Python
- Pandas
- Matplotlib
- Seaborn
## Analysis Performed
- Dataset inspection
- Summary statistics
- Scatter plot visualization
- Correlation heatmap
## Example Insight
The analysis shows that the living area of a house and the overall quality 
rating have strong relationships with house price.
## Purpose
This project demonstrates exploratory data analysis techniques used in 
data science to understand real-world datasets.
